# Release 2025.8

## Overview

Release 2025.8 brings significant improvements to security, reporting, and application performance. This release introduces multi-factor authentication for enhanced account protection, new capabilities for exporting and sharing reports, critical bug fixes for account recovery, and performance improvements that speed up search operations across the platform. This release was completed on August 15, 2025.

## New Features

| ID | Title | Description |
|---|---|---|
| US-1001 | Multi-Factor Authentication | Users can now secure their accounts using multi-factor authentication with email verification. When enabled, users can verify their identity using email verification codes in addition to their password, providing an additional layer of security to protect against unauthorized access. |
| US-1002 | Export Dashboard Reports | Users can now export dashboard reports directly to PDF format for offline access, sharing with stakeholders, and record keeping. This capability allows teams to generate professional reports that can be distributed and archived for compliance and documentation purposes. |

## Enhancements

| ID | Title | Description |
|---|---|---|
| US-1003 | Improved Search Performance | Search response times have been significantly improved across the platform. Users will experience faster results when searching for information, leading to improved productivity and a better overall user experience. |

## Bug Fixes

| ID | Title | Description |
|---|---|---|
| BUG-2001 | Password Reset Email Failure | Fixed a critical issue where password reset emails were not being delivered to some users. Users who forgot their passwords can now successfully receive password reset emails and recover their accounts. |
| BUG-2002 | Duplicate Notifications | Resolved an issue that caused duplicate notifications to be displayed to users. Users will no longer receive duplicate messages, resulting in a cleaner notification experience. |
