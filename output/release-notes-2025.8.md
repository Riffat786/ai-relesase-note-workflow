# Release Notes - Version 2025.8

**Release Date:** August 15, 2025

---

## Overview

We're excited to announce version 2025.8, a significant release focused on strengthening security and enhancing user productivity. This update introduces enterprise-grade multi-factor authentication capabilities, streamlines report sharing workflows, and delivers substantial performance improvements across the platform.

---

## New Features

### Multi-Factor Authentication (MFA)
**Enhanced Security for Your Account**

We've introduced comprehensive multi-factor authentication support to protect your account with an additional layer of security. You can now enable MFA using authenticator apps, SMS verification, or security keys—giving you flexibility in choosing the authentication method that works best for your organization. This enterprise-grade security feature helps prevent unauthorized access and meets compliance requirements for regulated industries.

**Benefits:**
- Choose from multiple authentication methods (authenticator apps, SMS, security keys)
- Seamless setup with step-by-step guidance
- Backup codes for account recovery
- Organization-wide MFA policies (for administrators)

### Export Dashboard Reports to PDF
**Share Insights with Ease**

Create professional PDF exports of your dashboards with a single click. Whether you need to share performance metrics with stakeholders, archive historical data, or include reports in presentations, you can now generate polished, ready-to-share documents directly from your dashboards.

**Benefits:**
- Export any dashboard configuration to a professional PDF
- Maintain formatting and visualizations
- Schedule recurring exports for automated distribution
- Share reports securely with customizable access controls

---

## Enhancements

### Improved Search Performance
**Find What You Need Faster**

We've optimized our search engine to deliver results in a fraction of the time. Whether you're searching across thousands of records or filtering complex datasets, you'll experience dramatically faster response times, allowing you to focus on insights instead of waiting for results.

**What's Improved:**
- 3x faster search response times on average
- Optimized indexing for large datasets
- Enhanced filtering capabilities
- Improved accuracy of search results

---

## Bug Fixes

### Password Reset Email Failure (Critical)
**Resolve: Users Unable to Reset Passwords**

We've fixed a critical issue that prevented users from successfully resetting their passwords via email. Users experiencing login difficulties can now reliably use the password reset flow to regain access to their accounts.

### Duplicate Notifications
**Resolve: Duplicate Messages and Alerts**

Fixed an issue that caused duplicate notifications to be sent to users. Notification delivery is now reliable and consistent, reducing notification fatigue and improving the user experience.

---

## Recommended Actions

**For All Users:**
- If you haven't already, we recommend enabling multi-factor authentication for enhanced security
- Explore the new PDF export feature for your dashboards to streamline reporting workflows

**For Administrators:**
- Review MFA configuration options to establish security policies for your organization
- Consider scheduling recurring dashboard exports for stakeholder reporting

---

## Known Limitations

- Dashboard PDF exports are currently limited to 20 pages; larger dashboards may need to be exported in sections
- Multi-factor authentication setup requires a valid mobile device or security key
- Search performance improvements apply primarily to indexed content; real-time data sources may vary
