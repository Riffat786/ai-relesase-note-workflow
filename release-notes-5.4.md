# Release Overview

**Version:** 5.4  
**Release Type:** Feature Release

This release introduces enhanced security capabilities and improved data management features, enabling users to better protect their accounts and streamline reporting workflows.

---

## New Features

### Multi-Factor Authentication (MFA)
Users can now configure multi-factor authentication using authenticator applications, adding an extra layer of protection to their accounts. This feature strengthens account security and helps prevent unauthorized access.

---

## Enhancements

### Dashboard Export to Excel
Export dashboard data directly to Microsoft Excel format, making it easier to share insights with stakeholders and conduct further analysis in familiar tools. This enhancement streamlines reporting workflows and improves data accessibility across your organization.

---

## Resolved Issues

### Report Export Failure
Fixed an issue where large report exports occasionally failed to complete successfully. Users can now reliably download reports of any size without interruption.

---

## Known Limitations

None identified for this release.
