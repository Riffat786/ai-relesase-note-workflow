# Create Executive Summary

## Purpose

Generate a concise executive-level summary of release content.

## Instructions

Use:

- Documentation Style Guide

Create a high-level summary suitable for management and stakeholders.

Focus on:

- Major features
- Business value
- Customer impact
- Risks
- Limitations

Avoid:

- Technical implementation details
- Detailed issue lists

## Input

Approved Release Notes

## Output Structure

# Executive Summary

## Key Highlights

## Customer Benefits

## Risks and Considerations
