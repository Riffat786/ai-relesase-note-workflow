# /show-workflow

## Purpose

Display the end-to-end AI workflow used to generate release notes.

---

## Usage

```text
/show-workflow
```

---

## Workflow

```text
Release Data
        │
        ▼
Collector Agent
        │
        ▼
Analyzer Agent
        │
        ▼
Writer Agent
        │
        ▼
Reviewer Agent
        │
        ▼
Publication Ready Output
```

---

## Agents Used

* Collector
* Analyzer
* Writer
* Reviewer

---

## Output

Displays the workflow diagram and a brief explanation of each stage.
