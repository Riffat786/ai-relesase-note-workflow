# /help

## Purpose

Display the available AI commands and their purpose.

---

## Usage

```text
/help
```

---

## Available Commands

| Command                   | Description             |
| ------------------------- | ----------------------- |
| `/collect-release-data`   | Retrieve release data   |
| `/generate-release-notes` | Generate release notes  |
| `/review-release-notes`   | Review release notes    |
| `/show-workflow`          | Display AI workflow     |
| `/help`                   | Show available commands |
| `/create-release-draft`   | Creates a draft in Doc360|

---

## Output

A summary of supported commands with a short description of each.
