# Release Overview

**Version:** 5.4  
**Release Type:** Feature Release

Version 5.4 introduces enhanced security capabilities, improved reporting functionality, and resolves critical issues. This release helps teams protect their data with stronger authentication methods while streamlining data analysis and sharing workflows.

---

## New Features

### Multi-Factor Authentication

Users can now enable multi-factor authentication using authenticator applications, adding a critical security layer to protect user accounts from unauthorized access. This feature is ideal for organizations managing sensitive data or operating in regulated environments requiring enhanced security controls.

**Key Benefits:**
- Prevents unauthorized account access even if passwords are compromised
- Supports industry-standard authenticator applications
- Recommended for all users, essential for administrators

---

## Enhancements

### Dashboard Export to Excel

Users can now export dashboard data directly to Microsoft Excel format, eliminating manual data transfer steps and enabling faster analysis workflows. This enhancement streamlines collaboration by allowing teams to share insights in familiar spreadsheet formats.

**Key Benefits:**
- Export dashboards with a single click
- Maintain data formatting during transfer
- Perform advanced analysis and create custom reports in Excel
- Simplify sharing with team members outside the application

---

## Resolved Issues

### Report Export Reliability

**Issue:** Large reports failed to export successfully, preventing users from accessing critical data in exportable formats.

**Resolution:** Fixed the underlying export mechanism to support reports of all sizes without timeout or corruption errors.

**Impact:** Users can now reliably download and share reports regardless of size, improving data accessibility and compliance workflows.

---

## Scope & Compatibility

These enhancements are available to all Version 5.4 users. Multi-factor authentication setup is optional but recommended for all user roles. No migration steps are required—existing reports and dashboards continue to function as before.
