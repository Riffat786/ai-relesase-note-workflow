# Release Overview

This release introduces new security capabilities, reporting improvements, and issue resolutions to enhance the overall user experience.

---

## New Features

### Multi-Factor Authentication

Users can now enable multi-factor authentication using authenticator applications, providing an additional layer of account security.

---

## Enhancements

### Dashboard Export to Excel

Users can now export dashboard data directly to Microsoft Excel, making it easier to share and analyze information.

---

## Resolved Issues

### Report Export Reliability

Resolved an issue that could prevent large reports from exporting successfully.
