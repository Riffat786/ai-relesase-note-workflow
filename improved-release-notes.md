# Release Overview

**Version:** 5.5  
**Release Type:** Mixed Release

---

## New Features

### Configuration Profiles
**For:** All Users

We've added support for configuration profiles, enabling you to create, save, and manage multiple configurations for different use cases and environments. This allows teams to quickly switch between pre-configured setups without manual reconfiguration.

---

## Security Updates

### Enhanced Password Policy Controls
**For:** Administrators and IT Teams

Administrators can now configure minimum password complexity requirements directly within the system, providing improved security controls and greater flexibility in enforcing organizational security standards across your deployment.

---

## Enhancements

None identified beyond security updates in this release.

---

## Resolved Issues

### Scheduled Report Reliability
**For:** Report Users and Administrators

Fixed an issue where scheduled reports occasionally failed when generated overnight, ensuring consistent and reliable report generation across all scheduling timeframes.
