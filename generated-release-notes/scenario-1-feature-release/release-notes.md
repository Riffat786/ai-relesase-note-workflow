# Release Notes - Version 5.4

## Release Overview

Version 5.4 introduces enhanced security capabilities, improved reporting functionality, and resolves critical issues to deliver a more secure and productive user experience.

---

## New Features

### Multi-Factor Authentication

Users can now enable multi-factor authentication using authenticator applications. This provides an additional layer of security to protect user accounts from unauthorized access.

---

## Enhancements

### Dashboard Export to Excel

Users can now export dashboard data directly to Microsoft Excel format. This makes it easier to share data with team members and perform additional analysis outside the application.

---

## Resolved Issues

### Report Export Reliability

Resolved an issue that prevented large reports from exporting successfully. Users can now reliably download reports of any size.

---

## Summary

This feature release focuses on security and reporting improvements that address key customer needs:
- **Security**: Multi-factor authentication for enhanced account protection
- **Productivity**: Direct Excel export capability for easier data sharing
- **Reliability**: Fixed export issues affecting large reports
