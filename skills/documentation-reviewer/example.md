# Example 1 - Ambiguous Wording

## Input

Improved system performance.

---

## Finding

Ambiguous statement.

Performance improvement is not described.

---

## Recommendation

Specify which user-facing performance improvements were made.

---

# Example 2 - Unsupported Claim

## Input

The platform is now significantly faster.

---

## Finding

Unsupported claim.

No evidence or measurable improvement provided.

---

## Recommendation

Provide measurable performance details or remove the claim.

---

# Example 3 - Missing Customer Impact

## Input

Added configuration profile support.

---

## Finding

Customer impact not identified.

---

## Recommendation

Explain how configuration profiles benefit users.

---

# Example 4 - Terminology Issue

## Input

End users can now setup MFA.

---

## Findings

1. Use "Users" instead of "End users".
2. Use "set up" instead of "setup" when used as a verb.

---

## Recommendation

Users can now set up multi-factor authentication.
